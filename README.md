# Aeries Grade Reports

Auto-generated grade reports from Aeries student portal.

## View Reports

**GitHub Pages:** [https://fiercestxdog.github.io/Aeries/](https://fiercestxdog.github.io/Aeries/)

## Available Reports

| Report | Description |
|--------|-------------|
| [Weekly Summary](weekly_summary.html) | AI-generated overview of grades and recent activity |
| [Missing Assignments](missing_report.html) | Assignments needing attention (score=0, past due) |
| [Grade Trends](grades_percentage.html) | Interactive Plotly chart of grade history |
| [Upcoming Assignments](upcoming_assignments.html) | Assignments due soon |

## Update Schedule

Reports are updated daily at 6:30 AM via local automation.
